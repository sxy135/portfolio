Flash Word

Flash Word is an application in HoloLens which could help us to memorize words in any language.

Type :  Individual Project for the Microsoft Beauty of Programming Competition
Award :   Excellent Work in the Creativity Competition of BOP
Time :   2016/05

Background

Flash Word is an individual project for the innovation competition of the Microsoft Beauty of Programming. The requirement of this competition was to demonstrate creative ideas about how Microsoft’s technology could be combined with other subjects. I got this idea when I was preparing for the GRE. I had a hard time memorizing English words. As a consequence, I tried to explore interesting ways to help students to memorize words.

Why HoloLens

According to many educational experts, the best way to memory English words is by memorizing them in the context. Also, it is universally acknowledged that 3D scenes can impress us much more than 2D pictures or texts.

Since HoloLens can build a mixed reality environment which combines VR and AR technology together, we could play the English words in a virtual reality scene to impress users and tag objects in the physical world with English words to help users to learn.

Learning Mode

When the user is in the learning mode, new words will appear in the corresponding virtual reality scene. The immersive scene will strengthen the memory and it can also make the learning process much more enjoyable.

The user can choose whether they have learned a particular word with natural gestures like nodding their head: then the word will disappear or repeat according to the user’s performance.

Daily Mode

The user can use daily mode in everyday life. With the augmented reality and computer vision technology, the physical objects in the horizon of HoloLens will be tagged with English words. Therefore, the user can revise the new words without interrupting daily life.

Product Extension

Users can edit their own scene to help them better memorize a particular word. Also, users can create new tags to help improve the performance of the application.

Besides, this application is not limited to English learning, it can be extended to other subjects as well.
